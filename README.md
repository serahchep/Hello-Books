# Prerequisites
  * pip
  * virtualenv
  * python 3 or python 2.7
  * flask
  * nosetests
# Installation
create a virtual environment using
* virtualenv
* install dependencies using

* $pip install -r requirements.txt
* pip install Flask

# Running the tests
tests are done using postman
